## Moddingとは

Idle Factoryは3種類のModに対応しており、それぞれ異なる方法でゲームを拡張できます。

| 種類 | 難易度 | できること |
|------|--------|-----------|
| **Data Mod** | 簡単 | アイテム・機械・レシピの追加 |
| **Script Mod** | 中級 | 外部ツール・オーバーレイの作成 |
| **Core Mod** | 上級 | ゲーム動作の変更 |

## 例: 最初のData Mod

たった2ファイルで新アイテムを作成:

**mods/my_mod/mod.toml**
```toml
[mod]
id = "my_mod"
name = "My Mod"
version = "1.0.0"

[mod.dependencies]
base = ">=0.3.0"
```

**mods/my_mod/items.toml**
```toml
[[item]]
id = "diamond"
name = "Diamond"
description = "A rare gem"
stack_size = 999
category = "ore"
```

ゲームを起動すると、ダイヤモンドアイテムが自動的に追加されます!

## ファイル構成

```
mods/
└── my_mod/
    ├── mod.toml        # 必須: Modメタデータ
    ├── items.toml      # 任意: アイテム定義
    ├── machines.toml   # 任意: 機械定義
    ├── recipes.toml    # 任意: レシピ定義
    ├── textures/       # 任意: PNGテクスチャ
    └── models/         # 任意: GLB/GLTF 3Dモデル
```
