# コンベアシステム

[English version](Conveyor-System)

自動アイテム輸送のためのコンベアベルトシステムの概要。

---

## 基本

コンベアはアイテムを自動的にある場所から別の場所に移動します。

### 設置方法

1. ホットバーからコンベアを選択
2. クリックして設置
3. コンベアは自動的に接続

### 方向

コンベアには矢印で示される方向があります。

設置前にRキーで回転。

---

## コンベアの形状

| 形状 | 説明 |
|-----|------|
| 直線 | 直接パス |
| コーナー（左/右） | 90度ターン |
| T字路 | 分岐/合流点 |
| スプリッター | 1入力 → 2出力 |

---

## 機械との接続

### 入力接続

コンベアを機械の入力ポートに接触させて設置。

```
[コンベア] → [機械入力ポート]
```

### 出力接続

コンベアを機械の出力ポートに設置。

```
[機械出力ポート] → [コンベア]
```

---

## アイテムの流れ

アイテムはコンベア上を一定速度で移動します。

### ブロッキング

目的地がいっぱいの場合、アイテムは停止します。

### 優先度

合流時、アイテムは入力間で交互になります。

---

## 自動化の例

基本的な鉱石加工ライン：

```
[採掘機] → [コンベア] → [粉砕機] → [コンベア] → [精錬炉] → [コンベア] → [倉庫]
```

---

## ヒント

### ラインは短く

短いコンベアラインはスループットが速い。

### ループを避ける

ループはアイテムが永遠に循環する原因になる。

### スプリッターを使う

スプリッターでアイテムを複数の機械に分配して並列処理。
