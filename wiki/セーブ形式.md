# セーブ形式

[English version](Save-Format)

セーブファイル形式の技術ドキュメント。

---

## 概要

セーブは `saves/` ディレクトリにJSON形式で保存されます。

---

## ファイルの場所

| プラットフォーム | パス |
|----------------|------|
| Windows | `%APPDATA%/idle_factory/saves/` |
| Linux | `~/.local/share/idle_factory/saves/` |
| macOS | `~/Library/Application Support/idle_factory/saves/` |

---

## フォーマットバージョン

現在のフォーマット: **V2**（文字列ベースのItemId）

---

## 構造

```json
{
  "version": 2,
  "player": {
    "position": [x, y, z],
    "inventory": [...]
  },
  "world": {
    "chunks": {...},
    "machines": [...]
  },
  "quests": {...}
}
```

---

## Mod互換性

Modアイテムを使用するセーブは以下の場合に互換性があります：

- Modがまだインストールされている
- アイテムIDが変更されていない

不明なアイテムはロード時にログに記録されスキップされます。
