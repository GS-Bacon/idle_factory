# ゲーム内テストシナリオ実行

ゲーム内の `/test` コマンドを使用して、自動テストシナリオを実行する。

## 引数
$ARGUMENTS

## 概要

このスキルは、ゲーム内でテストシナリオを自動実行し、機械チェーンやアイテムフローが正しく動作するかを検証します。

## 使用方法

### 1. ゲームを起動

```bash
./run.sh
```

### 2. コマンド入力

ゲーム内で `T` または `/` キーを押してコマンドバーを開き、以下を入力:

```
/test <scenario>
```

## 利用可能なシナリオ

| シナリオ | 内容 |
|----------|------|
| `miner_chain` | Miner → Conveyor → Furnace の接続テスト |
| `conveyor_flow` | コンベア上のアイテム移動テスト |
| `furnace_smelt` | 精錬炉の入力→出力テスト |
| `full_automation` | 採掘→加工→納品の全自動化テスト |
