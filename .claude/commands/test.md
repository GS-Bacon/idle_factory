# ゲーム内テストシナリオ実行

ゲーム内の `/test` コマンドを使用して、自動テストシナリオを実行する。

## 引数
$ARGUMENTS

## 概要

このスキルは、ゲーム内でテストシナリオを自動実行し、機械チェーンやアイテムフローが正しく動作するかを検証します。

## 使用方法

### 1. ゲームを起動

```bash
./run.sh
```

### 2. コマンド入力

ゲーム内で `T` または `/` キーを押してコマンドバーを開き、以下を入力:

```
/test <scenario>
```

## 利用可能なシナリオ

| シナリオ | 内容 |
|----------|------|
| `production` | Miner → 3x Conveyor → Furnace ライン配置（石炭16個付与） |
| `stress` | 10x10 コンベアグリッド（100個）でストレステスト |
