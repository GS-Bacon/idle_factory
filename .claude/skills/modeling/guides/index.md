# モデリングガイド一覧

## 使い方

1. モデルを作る前に該当ガイドを参照
2. `/review-models` でHTMLプレビュー作成
3. デザイン確定後にBlenderでモデリング

## 共通ルール

### サイズ基準
- 1ブロック = 1m x 1m x 1m
- アイテム = 0.1〜0.15m程度

### 色パレット

| 用途 | 色 |
|------|-----|
| 金属（暗） | #333333 〜 #555555 |
| 金属（明） | #888888 〜 #aaaaaa |
| 鉄 | #c0c0c0 |
| 銅 | #b87333 |
| 木 | #8b4513 |
| 石 | #808080 |
| インジケーター（緑） | #00ff00 |
| インジケーター（青） | #0066ff |
| インジケーター（黄） | #ffcc00 |
| 警告（赤） | #ff0000 |

### ポート色

| タイプ | 色 |
|--------|-----|
| 入力 | 緑 #00aa00 |
| 出力 | 赤/オレンジ #aa0000 |
| 燃料 | 黄 #aaaa00 |
