# コンベア (Conveyor) モデリングガイド

## 概要

| 項目 | 値 |
|------|-----|
| ID | `conveyor` |
| 日本語名 | コンベア |
| サイズ | 1ブロック (1m x 0.3m x 1m) |
| 機能 | アイテムを自動搬送 |

## デザインコンセプト

- **ベルトコンベア**
- フレーム（両サイドのレール）
- 動くベルト（中央）
- 低い高さ（0.2〜0.3m）

## 構造

```
   [Rail]═════════[Rail]
     ║   [Belt]     ║
     ║ ▶▶▶▶▶▶▶▶▶▶  ║   ← ベルト: 搬送方向
     ║═════════════║
   [Leg]         [Leg]   ← 脚: 4本（角）
```

## パーツ詳細

| パーツ | 形状 | 推奨サイズ | 色 |
|--------|------|-----------|-----|
| Frame Left | ボックス | 0.05 x 0.15 x 1.0 | 暗灰色 #3a3a3a |
| Frame Right | ボックス | 0.05 x 0.15 x 1.0 | 暗灰色 #3a3a3a |
| Belt | ボックス | 0.8 x 0.05 x 1.0 | 茶色 #4a3a2a |
| Legs | 細ボックス x4 | 0.03 x 0.1 x 0.03 | 灰色 #555555 |
| Roller | 円柱 x2 | 直径0.06 | 金属色 #666666 |

## I/Oポート

| ポート | 位置 | 方向 |
|--------|------|------|
| 入力 | 後面 (Back) | アイテム受け取り |
| 出力 | 前面 (Front) | アイテム排出 |

## 方向表示

- ベルトの矢印/テクスチャで搬送方向を示す
- 端のローラーで入出力を区別
