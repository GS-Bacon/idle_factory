# 納品プラットフォーム (Platform) モデリングガイド

## 概要

| 項目 | 値 |
|------|-----|
| ID | `platform` |
| 日本語名 | 納品プラットフォーム |
| サイズ | 1ブロック (1m x 0.5m x 1m) |
| 機能 | アイテムを収納、クエスト納品 |

## デザインコンセプト

- **収納パレット/プラットフォーム**
- 低い台座
- アイテムが積み重なるイメージ
- 光るビーコン（目印）

## 構造

```
       [Beacon]      ← ビーコン: 目印の光
    ╔═══════════╗
    ║  ▢ ▢ ▢   ║    ← 上面: アイテム積載エリア
    ║  Platform ║
    ╠═══════════╣
    ║   Base    ║    ← ベース: 支持構造
    ╚═══════════╝
```

## パーツ詳細

| パーツ | 形状 | 推奨サイズ | 色 |
|--------|------|-----------|-----|
| Base | ボックス | 0.9 x 0.15 x 0.9 | 金属色 #4a4a4a |
| Platform | ボックス | 0.85 x 0.05 x 0.85 | 木色 #8b7355 |
| Beacon | 小円柱 | 直径0.05 x 高さ0.2 | 黄光 #ffcc00 |
| Edge Rails | 細ボックス x4 | 0.02 x 0.1 x 0.85 | 金属色 #555555 |

## I/Oポート

| ポート | 位置 | 方向 |
|--------|------|------|
| 入力 | 全方向 | アイテム受け取り |

## 状態表現

| 状態 | 表現 |
|------|------|
| 空 | ビーコン暗い |
| アイテムあり | ビーコン光る |

## 参考デザイン設定 (JSON)

```json
{
  "baseWidth": 0.9,
  "baseHeight": 0.15,
  "baseColor": "#4a4a4a",
  "platformWidth": 0.85,
  "platformHeight": 0.05,
  "platformColor": "#8b7355",
  "beaconRadius": 0.025,
  "beaconHeight": 0.2,
  "beaconColor": "#ffcc00",
  "railWidth": 0.02,
  "railHeight": 0.1
}
```
