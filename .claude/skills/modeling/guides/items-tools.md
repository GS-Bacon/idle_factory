# ツールアイテム モデリングガイド

## 概要

プレイヤーが使用するツール。手持ち表示あり。

| アイテム | ID | 用途 |
|----------|-----|------|
| 石のツルハシ | `stone_pickaxe` | ブロック破壊 |

## デザインコンセプト

- **マインクラフト風のシンプルなツルハシ**
- ローポリ
- 柄と頭部の2パーツ
- 手持ち時に自然に見える向き

## 構造

```
      ╔═══╗
    ═╣Head╠═        ← 頭部: 石製
      ╚═╤═╝
        │
        │           ← 柄: 木製
        │
       ═╧═          ← 柄の端
```

## サイズ

| 用途 | サイズ |
|------|--------|
| 手持ち表示 | 全長 0.8m |
| インベントリ | アイコン用にレンダリング |

## 手持ち表示の向き

```
プレイヤー視点:
     ╔═══╗
   ═╣    ╠═   ← 頭部は右向き
     ╚═╤═╝
       │
       │      ← 柄は下向き
      ─┘
     (手)
```

- 頭部がプレイヤーの右側に向く
- 柄を握る位置は下から1/3程度
