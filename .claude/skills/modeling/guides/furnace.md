# 精錬炉 (Furnace) モデリングガイド

## 概要

| 項目 | 値 |
|------|-----|
| ID | `furnace` |
| 日本語名 | 精錬炉 |
| サイズ | 1ブロック (1m x 1m x 1m) |
| 機能 | 鉱石をインゴットに精錬（燃料必要） |

## デザインコンセプト

- **工業用溶鉱炉**
- 煙突（上部）
- 炉口（前面、光る）
- 燃料投入口（側面）
- 頑丈で重厚なデザイン

## 構造

```
       ╔═╗
       ║煙║        ← 煙突: 上部から煙が出る
    ┌──╨─╨──┐
    │       │
    │  ███  │      ← 炉口: 前面（稼働中は光る）
    │ [Body]│
燃料→│      │←燃料  ← 側面: 燃料入力
    └───────┘
        ↓          ← 前面下部: 出力
```

## パーツ詳細

| パーツ | 形状 | 推奨サイズ | 色 |
|--------|------|-----------|-----|
| Body | ボックス | 0.8 x 0.7 x 0.8 | レンガ色 #8b4513 |
| Chimney | 円柱/八角柱 | 直径0.15 x 高さ0.3 | 暗灰色 #333333 |
| Door | ボックス（凹み） | 0.3 x 0.25 | 黒/オレンジ(稼働時) |
| Fuel Port | 小ボックス x2 | 0.15 x 0.15 | 金属色 #555555 |
| Base | ボックス | 0.9 x 0.1 x 0.9 | 石色 #4a4a4a |

## I/Oポート

| ポート | 位置 | 方向 |
|--------|------|------|
| 入力(鉱石) | 後面 (Back) | 素材入力 |
| 入力(燃料) | 左側 (Left) | 燃料入力 |
| 入力(燃料) | 右側 (Right) | 燃料入力 |
| 出力 | 前面 (Front) | 精錬結果出力 |

## 状態表現

| 状態 | 表現 |
|------|------|
| 停止 | 炉口が暗い |
| 稼働中 | 炉口がオレンジに光る、煙が出る |
