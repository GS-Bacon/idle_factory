# Idle Factory スタイルガイド

採掘機をリファレンスとした統一デザイン言語。

---

## カラーパレット

| 用途 | カラー | Hex |
|------|--------|-----|
| 本体（メイン） | ダークグレー | `#4a4a4a` |
| 脚・フレーム | ミディアムグレー | `#555555` |
| シャフト・排出口 | グレー | `#666666` |
| ドリル・加工部 | ライトグレー | `#888888` |
| **インジケーター** | **緑** | `#00ff00` |
| アクセント（熱） | オレンジ | `#ff6600` |
| アクセント（電気） | 青 | `#00aaff` |

### ルール

- **モノトーン基調** - グレースケールで統一感
- **インジケーターで機能表示** - 緑=稼働、オレンジ=加熱、青=電気
- **派手な色は小さく** - アクセントは面積1-5%

---

## スケール

| 項目 | 値 | 備考 |
|------|-----|------|
| 1ブロック | 1.0 x 1.0 x 1.0 | ゲーム内の基本単位 |
| **modelScale** | **1.5〜1.8** | 基本設計を拡大して1ブロックに収める |
| はみ出し | 禁止 | 当たり判定と見た目を一致させる |

### 基準（採掘機）

```
modelScale: 1.7
実効サイズ: 約0.9ブロック（余白あり）
```

## 構造パターン

### 基本構成

```
[上部]    モーター/加工部/排気口
[中部]    本体（メインボックス）
[下部]    脚/ベース/接地部
```

### 本体比率（採掘機基準）

| パーツ | 幅 | 高さ | 奥行 | 備考 |
|--------|-----|------|------|------|
| 本体 | 0.31 | 0.16 | 0.32 | ほぼ正方形、薄め |
| 脚広がり | 0.34 | - | - | 安定感重視 |
| 脚太さ | 0.04 | - | - | 細すぎない |

## デザイン原則

### DO（やる）

- **機能が見える** - 何をする機械か一目でわかる
- **重厚感** - 工業機械らしい存在感
- **統一感** - 同じ工場に並んで違和感がない
- **脚は広げる** - 安定感、接地感
- **パーツを重ねる** - 接触ではなく重なりで一体感

### DON'T（やらない）

- **派手な色** - 彩度の高い色を大面積で使わない
- **浮遊感** - 脚なし、ベースなしは避ける
- **過剰なディテール** - Low-polyの範囲で
- **角ばりすぎ** - 面取り・丸みを適度に
- **はみ出し** - 当たり判定ボックスを超えない

## Three.js → Blender変換時の注意

| 項目 | Three.js | Blender |
|------|----------|---------|
| 上方向 | Y | Z |
| 座標変換 | (x, y, z) | (x, z, y) |
| スケール | そのまま | modelScale適用済み |

## クイックリファレンス

### 新しい機械を作る時

1. 採掘機の `mining_drill_preview.html` を複製
2. パーツ構成を変更（ドリル→炎窓など）
3. カラーパレットに従って色設定
4. modelScale 1.5〜1.8 で調整
5. 当たり判定ボックスに収まるか確認
6. エクスポート
