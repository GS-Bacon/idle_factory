## CRITICAL (ゲームプレイに支障)

### BUG-V001: 地形が真っ黒に表示される

**発見場所**: `e2e_full/11_native_placed_machines.png`, `e2e_full/12_native_overview.png`

**症状**:
- 俯瞰視点で地形が完全に黒く表示される
- コンベア/機械のみが小さなドットとして見える
- ワールドの地面が描画されていない

**影響**:
- プレイヤーが地形を認識できない
- ブロック配置位置がわからない
- ゲームプレイが困難

**推定原因**:
1. カメラが地面より下にある
2. 地形メッシュのカリング問題
3. ライティングが届いていない

**確認方法**:
```bash
./scripts/vlm_check.sh --chunk
```
