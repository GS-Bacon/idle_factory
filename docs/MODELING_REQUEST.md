# モデリング依頼書

## 概要

Idle Factory ゲーム用の3Dモデル作成依頼です。
MagicaVoxel (.vox) で作成してください。

## スタイルガイド

- **テイスト**: Minecraft / Unturned風のブロック感
- **テクスチャ**: なし（マテリアルカラーのみ、Astroneer風）
- **単位**: 16ボクセル = 1ゲームブロック

## 機械モデル (16x16x16 voxels)

### 1. 採掘機 (miner.vox)

**機能**: 地面の鉱石を自動で掘り出す

**デザイン指示**:
- 底面: dark_steel のベースプレート
- 本体: miner_body (オレンジ系) の立方体
- 上部: ドリルヘッド (dark_steel + iron)
- 側面: 出力口（1面だけ穴が開いている）
- アクセント: 稼働ランプ (active) を1-2ドット

**イメージ**: 小型の自動掘削機

---

### 2. 精錬炉 (furnace.vox)

**機能**: 鉱石をインゴットに精錬

**デザイン指示**:
- 本体: furnace_body (レンガ茶) のブロック
- 前面: 炉の開口部 (furnace_glow) 2x2ドット
- 上部: 煙突 (dark_steel) 4x4x4
- 側面: 入力口と出力口を示す

**イメージ**: 溶鉱炉、レンガ造り

---

### 3. 粉砕機 (crusher.vox)

**機能**: 鉱石を細かく砕く

**デザイン指示**:
- 本体: crusher_body (紫系) のブロック
- 上部: ホッパー状の投入口 (dark_steel)
- 中央: 歯車/ローラー (iron) が見える
- 下部: 出力シュート
- アクセント: warning (黄) のストライプ

**イメージ**: 工業用クラッシャー

---

### 4. 組立機 (assembler.vox)

**機能**: 部品を組み合わせる

**デザイン指示**:
- 本体: dark_steel フレーム
- 中央: brass の作業台
- 上部: ロボットアーム風の突起
- 側面: 複数の入力スロット
- アクセント: active ランプ

**イメージ**: 自動組立ロボット

---

### 5. 保管庫 (storage.vox)

**機能**: アイテムを貯蔵

**デザイン指示**:
- 本体: dark_steel の大きな箱
- 前面: 格子状の扉 (iron)
- 側面: 換気口のスリット
- 角: brass の補強

**イメージ**: 工業用倉庫

---

### 6. 発電機 (generator.vox)

**機能**: 電力を生成

**デザイン指示**:
- 本体: dark_steel
- 中央: power (緑) のコア
- 側面: 冷却フィン (iron)
- 上部: 電極 (brass)
- アクセント: 稼働時は active ランプ

**イメージ**: 発電ユニット

---

### 7. インサータ (inserter.vox)

**機能**: アイテムを隣のマシンに移動

**デザイン指示**:
- ベース: 4x4x4 の dark_steel 台座
- アーム: iron の細いアーム (3-4ドット幅)
- 先端: brass のグリッパー
- 高さ: 12-14ドット（16に収まる）

**イメージ**: ロボットアーム、Factorio風

---

### 8. 分配機 (splitter_machine.vox)

**機能**: アイテムを複数方向に分配

**デザイン指示**:
- 本体: frame カラーのボックス
- 上部: 投入口
- 側面: 3-4方向に出力口
- 内部: 回転するセパレーター (iron)
- アクセント: warning ストライプ

**イメージ**: アイテム分岐器

---

## コンベアモデル (16x12x4 voxels)

高さ4ドットの薄い形状。上面がベルト。

### 1. 直線コンベア (conveyor/straight.vox)

**デザイン指示**:
- 左右: frame のフレーム (2ドット幅)
- 中央: belt の表面
- ベルト上: 進行方向を示す矢印 (roller で1ドット)
- ローラー: 両端に roller の軸

**イメージ**: → 方向に流れる

---

### 2. 左カーブ (conveyor/corner_left.vox)

**デザイン指示**:
- frame のL字フレーム
- belt で90度カーブ
- 角は丸くせず、ボクセルらしく階段状

**イメージ**: ↑ から ← へ曲がる

---

### 3. 右カーブ (conveyor/corner_right.vox)

**デザイン指示**:
- corner_left の左右反転

**イメージ**: ↑ から → へ曲がる

---

### 4. T字分岐 (conveyor/t_junction.vox)

**デザイン指示**:
- 3方向にベルトが伸びる
- 中央で合流/分岐

**イメージ**: ⊥ 形状

---

### 5. 分配器 (conveyor/splitter.vox)

**デザイン指示**:
- 入力1、出力2 の分岐器
- 中央に小さな分配ブロック

---

## アイテムモデル (8x8x8 voxels)

コンベア上で流れる小さなアイテム。

### 1. 鉄鉱石 (items/iron_ore.vox)

**デザイン指示**:
- ゴツゴツした岩の形
- stone ベースに iron の斑点
- 不規則な形状（完全な立方体でない）

---

### 2. 銅鉱石 (items/copper_ore.vox)

**デザイン指示**:
- 鉄鉱石と同様の形状
- stone ベースに copper の斑点

---

### 3. 石炭 (items/coal.vox)

**デザイン指示**:
- 黒い塊 (belt カラー = #303030)
- 角張った形状
- わずかに光沢 (dark_steel のハイライト)

---

### 4. 石 (items/stone.vox)

**デザイン指示**:
- stone カラーの塊
- 丸みを帯びた不規則形状

---

### 5. 鉄インゴット (items/iron_ingot.vox)

**デザイン指示**:
- iron カラー
- インゴット形状: 8x4x2 の台形
- 滑らかで整った形

---

### 6. 銅インゴット (items/copper_ingot.vox)

**デザイン指示**:
- copper カラー
- 鉄インゴットと同じ形状

## ブロックテクスチャアトラス

全ブロックテクスチャを **1枚の画像** にまとめます（パフォーマンス向上）。

### 保存先

```
assets/textures/block_atlas.png   # 512x512px (8x8グリッド、各16x16px)
```

### アトラス仕様

| 項目 | 値 |
|------|-----|
| 全体サイズ | 512x512 ピクセル |
| グリッド | 8x8 (64スロット) |
| 各ブロック | 16x16 ピクセル |
| 形式 | PNG (透過なし) |
| スタイル | Minecraft風ピクセルアート |

### アトラスレイアウト


### 各ブロックのデザイン指示

#### ID 0: 草ブロック上面 (0, 0)
- 明るい緑 (#4CAF50) ベース
- 少し暗い緑のノイズ/斑点
- Minecraft風の自然なランダムパターン

#### ID 1: 草ブロック側面 (1, 0)
- 上部2-3px: 草（緑 #4CAF50）
- 下部: 土（茶色 #8B6914）
- 境界は少しギザギザに

#### ID 2: 土 (2, 0)
- 茶色 (#8B6914) ベース
- 暗い茶色の斑点でテクスチャ感
- 小さな石粒（グレー）を数個

#### ID 3: 石 (3, 0)
- グレー (#808080) ベース
- 暗いグレーと明るいグレーの斑点
- ひび割れ風のライン（1px）

#### ID 4: 鉄鉱石 (4, 0)
- 石をベースに
- 鉄の斑点 (#C0C0C8) を5-8個散らす
- 斑点サイズ: 2x2 または 3x2 px

#### ID 5: 銅鉱石 (5, 0)
- 石をベースに
- 銅の斑点 (#E08050) を5-8個散らす
- 斑点サイズ: 2x2 または 3x2 px

#### ID 6: 石炭 (6, 0)
- 石をベースに
- 石炭の斑点 (#303030) を5-8個散らす
- 斑点は少し大きめ (3x3 px)

#### ID 7: 岩盤 (7, 0)
- 非常に暗いグレー (#202020) ベース
- 黒とダークグレーの不規則パターン
- 「破壊不可」な硬さを表現

#### ID 8: 砂 (0, 1)
- 黄色系 (#E0C080) ベース
- 明るい黄色の細かいノイズ
- 砂粒感

## 優先順位

### モデル
1. **高**: 採掘機、精錬炉、直線コンベア、鉄鉱石、鉄インゴット
2. **中**: 粉砕機、組立機、カーブコンベア、銅系アイテム
3. **低**: 発電機、保管庫、インサータ、分配機

### テクスチャ
1. **高**: grass_top, grass_side, stone, dirt
2. **中**: iron_ore, copper_ore, coal_ore
3. **低**: bedrock, sand
