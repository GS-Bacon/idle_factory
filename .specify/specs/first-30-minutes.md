# 最初の30分のプレイフロー

## 概要

プレイヤーに「手動→自動化」の対比を体験させ、自動化の快感を早期に提供する。

## 初期状態

| 項目 | 内容 |
|------|------|
| 初期装備 | ピッケル、精錬炉×1 |
| 初期位置 | 鉄鉱石・石炭が近くにあるスポーン地点 |
| 納品プラットフォーム | 12×12、初期16ポート開放 |

## Step 1: 手動フェーズ（1-2分）

**目的**: 基本操作と精錬の仕組みを学ぶ（短く！）

1. チュートリアルポップアップで操作説明
2. **スポーン地点に鉄鉱石×5、石炭×5が落ちている**（採掘不要）
3. 精錬炉に投入
   - 石炭 = 燃料スロット
   - 鉄鉱石 = 入力スロット
   - 鉄インゴット = 出力
4. 鉄インゴットをプラットフォームに納品

**クエスト**: 「鉄インゴットを3個納品せよ」（すぐ終わる量）

### 手動フェーズの設計意図

| 従来案 | 改善案 | 理由 |
|--------|--------|------|
| 5分間手動 | 1-2分 | 長すぎると離脱リスク |
| 鉱石を掘る | 落ちている | 「掘る」は自動化後でOK |
| 5個納品 | 3個納品 | 最低限で対比を体感 |

## Step 2: 報酬で採掘機入手（即時）

**報酬内容**:
- 採掘機×2
- コンベア×20
- **ガイドマーカー表示**（設置推奨位置をハイライト）

**演出**: 「これで自動化できるぞ！」的なヒント表示

## Step 3: 自動化フェーズ（10-15分）

**目的**: 採掘→精錬→納品の自動ラインを構築

1. 採掘機を鉄鉱石バイオームに設置
   - ガイドマーカーに従えば迷わない
   - 採掘機は資源バイオーム上で無限採掘
2. コンベアで精錬炉の入力に接続
3. 別の採掘機を石炭バイオームに設置
4. コンベアで精錬炉の燃料入力に接続
5. 精錬炉の出力からプラットフォームへコンベア接続
6. 自動納品ラインの完成！

**クエスト**: 「鉄インゴットを100個納品せよ」

### ガイドマーカー

初回プレイ時のみ表示される設置ガイド。

| 表示 | 内容 |
|------|------|
| 緑の枠 | 「ここに採掘機を置こう」 |
| 青の線 | 「ここにコンベアを引こう」 |
| 黄のアイコン | 「精錬炉の入力はここ」 |

**設定でOFF可能**（ベテランプレイヤー向け）

## Step 4: スケールアップ（10分）

**報酬内容**:
- 採掘機×2
- コンベア×40
- 精錬炉×2

**目的**: 拡張の楽しさを体験

- 2ライン目を構築
- 「数を増やせばもっと速い」を実感
- 次のクエストへ（銅インゴット納品など）

## 設計意図

| ポイント | 理由 |
|----------|------|
| 手動フェーズは最小限 | 自動化ゲームなのに手動が長いのは本末転倒 |
| 素材は落ちている | 最初から「掘る」必要はない |
| ガイドマーカー | 初心者が迷わない、ベテランはOFFに |
| 報酬で機械を渡す | クラフトの手間を省く |
| 燃料が必要 | 資源管理の概念を早期導入 |

## 採掘機の仕様

| 項目 | 仕様 |
|------|------|
| 設置条件 | 資源バイオーム上に設置 |
| 採掘量 | 無限（資源枯渇なし） |
| 出力 | 1方向、コンベア接続可能 |

## ピッケルの用途

- 地形を掘る（建築スペース確保）
- ブロックを回収する
- 機械の撤去

※ 資源確保のための採掘は不要（採掘機が無限採掘）

## 2周目以降のオプション

### クイックスタートモード

ワールド作成時に選択可能。

| モード | 内容 |
|--------|------|
| 通常 | チュートリアル付き、ガイドマーカーあり |
| クイックスタート | チュートリアルスキップ、Tier 1まで解放済み |
| フリーモード | 全ティア解放、無限資材 |

### クイックスタートの初期状態

| 項目 | 内容 |
|------|------|
| 解放済み | Tier 0-1（採掘機、コンベア、精錬炉） |
| 初期インベントリ | 採掘機×4、コンベア×100、精錬炉×4 |
| クエスト | Step 3（100個納品）から開始 |

### フリーモードの初期状態

| 項目 | 内容 |
|------|------|
| 解放済み | 全ティア |
| 資材 | クリエイティブインベントリ（無限） |
| クエスト | 任意で進行可能 |
| 用途 | 設計実験、MOD開発、配信者向け |

### 設計意図

| ポイント | 理由 |
|----------|------|
| 2周目は強制しない | 「知ってる」のに繰り返すストレス回避 |
| フリーモードを用意 | クリエイティブ志向プレイヤー向け |
| 通常モードがデフォルト | 初心者には段階的体験が重要 |
