# 最初の30分のプレイフロー

## 概要

プレイヤーに「手動→自動化」の対比を体験させ、自動化の快感を早期に提供する。

## 初期状態

| 項目 | 内容 |
|------|------|
| 初期装備 | ピッケル、精錬炉×1 |
| 初期位置 | 鉄鉱石・石炭が近くにあるスポーン地点 |
| 納品プラットフォーム | 12×12、初期16ポート開放 |

## Step 1: 手動フェーズ（5分）

**目的**: 基本操作と精錬の仕組みを学ぶ

1. 鉄鉱石を手で拾う（地面に落ちている or ブロックを掘る）
2. 石炭を手で拾う
3. 精錬炉に投入
   - 石炭 = 燃料スロット
   - 鉄鉱石 = 入力スロット
   - 鉄インゴット = 出力
4. 鉄インゴットをプラットフォームに納品

**クエスト**: 「鉄インゴットを5個納品せよ」

## Step 2: 報酬で採掘機入手（1分）

**報酬内容**:
- 採掘機×2
- コンベア×20

**演出**: 「これで自動化できるぞ！」的なヒント表示

## Step 3: 自動化フェーズ（15分）

**目的**: 採掘→精錬→納品の自動ラインを構築

1. 採掘機を鉄鉱石バイオームに設置
   - 採掘機は資源バイオーム上で無限採掘
2. コンベアで精錬炉の入力に接続
3. 別の採掘機を石炭バイオームに設置
4. コンベアで精錬炉の燃料入力に接続
5. 精錬炉の出力からプラットフォームへコンベア接続
6. 自動納品ラインの完成！

**クエスト**: 「鉄インゴットを100個納品せよ」

## Step 4: スケールアップ（10分）

**報酬内容**:
- 採掘機×2
- コンベア×40
- 精錬炉×2

**目的**: 拡張の楽しさを体験

- 2ライン目を構築
- 「数を増やせばもっと速い」を実感
- 次のクエストへ（銅インゴット納品など）

## 設計意図

| ポイント | 理由 |
|----------|------|
| 手動フェーズを入れる | 自動化の「楽になった！」感を演出 |
| 報酬で機械を渡す | 進捗に応じた報酬ループの導入 |
| 燃料が必要 | 資源管理の概念を早期導入 |
| 報酬が初期装備と被る | 意図的。数を増やしてスケールアップを促す |

## 採掘機の仕様

| 項目 | 仕様 |
|------|------|
| 設置条件 | 資源バイオーム上に設置 |
| 採掘量 | 無限（資源枯渇なし） |
| 出力 | 1方向、コンベア接続可能 |

## ピッケルの用途

- 地形を掘る（建築スペース確保）
- ブロックを回収する
- 機械の撤去

※ 資源確保のための採掘は不要（採掘機が無限採掘）
