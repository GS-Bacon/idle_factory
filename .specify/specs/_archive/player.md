# プレイヤー

## 概要

サバイバル要素を排除した、工場建設に特化したプレイヤー仕様。

## ステータス

| 項目 | 値 |
|------|-----|
| HP | なし（無敵） |
| 飢餓 | なし |
| 落下ダメージ | なし |
| リスポーン | 即座 |

## インベントリ

| 項目 | 値 |
|------|-----|
| スロット数 | 40 |
| スタック上限 | 999 |
| ホットバー | 10スロット |

## 移動

| 手段 | 速度 | 入手時期 |
|------|------|----------|
| 徒歩 | 基本 | 初期 |
| ジェットパック | 高速、3D自由移動 | 初期〜早期 |
| ハイパーチューブ | 超高速、固定ルート | 中盤 |
| 列車 | 大量輸送と移動兼用 | 中盤 |
| テレポーター | 瞬間移動 | 終盤 |

## ツール

| ツール | 用途 |
|--------|------|
| ピッケル | 地形を掘る、ブロック回収 |
| レンチ | 機械の回転・設定変更 |
| ハンマー | 機械の撤去 |

## 設計思想

**なぜサバイバル要素を排除するか？**

- 工場建設に集中させたい
- 飯や敵の対処は本質的な楽しさではない
- 大規模工場での長距離移動ストレスを軽減（ジェットパック）
