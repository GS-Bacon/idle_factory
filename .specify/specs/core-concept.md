# コアコンセプト

## ゲーム概要

| 項目 | 内容 |
|------|------|
| ジャンル | 3Dボクセル工場シミュレーション |
| モード | クリエイティブサンドボックス（サバイバル要素なし） |
| 最終目標 | 軌道上の宇宙ステーション建造 |

## 差別化ポイント（既存ゲームへの不満→解決策）

| 既存ゲームの不満 | 本ゲームの解決策 |
|------------------|------------------|
| 序盤は工場より飯の心配 | HP・敵・飢餓・落下ダメージなし |
| 大規模工場での移動が遅い | ジェットパック標準装備 |
| コンベアの合流・分配挙動が雑 | ジッパー合流、均等分配にこだわり |
| アップグレードで破壊→再設置 | その場で置き換え可能 |
| 自動化に限界がある | 高度な自動化システム（将来検討） |
| 2D/レイヤー制限 | 3Dボクセル空間で完全自由配置 |
| Mod管理が面倒 | Mod SDK内蔵、自動読み込み |

## 楽しさの核

1. **拡張の満足感** - 工場がどんどん大きくなる達成感
2. **創造表現** - 3D空間で自由に美しいラインを構築
3. **眺める快感** - 動く工場を見る楽しさ（Shapez的）

## インスピレーション

| ゲーム | 取り入れる要素 |
|--------|---------------|
| Factorio | 自動化の深さ |
| Satisfactory | 3D工場建設の楽しさ |
| Shapez / Shapez 2 | 眺める快感、納品プラットフォーム |
| Minecraft (Create Mod) | ボクセルの自由度、全レシピ解放 |

## ターゲットプレイヤー

- ストレスフリーで純粋に工場だけ作りたい人
- コンベアの挙動（合流・分配）にこだわりたい人
- 複雑な自動化システムを組みたい人
- 3Dボクセル空間で美しいラインを構築したい人
- 大規模工場をジェットパックで飛び回りながら眺めたい人
