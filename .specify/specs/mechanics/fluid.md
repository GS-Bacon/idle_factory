# パイプ・液体システム

## 概要

液体・ガスの輸送と処理。コンベアの液体版。

## 単位

| 単位 | 説明 |
|------|------|
| mB（ミリバケツ） | 液体の量。1バケツ = 1000mB |
| mB/s | 流量。1秒あたりの移動量 |

## パイプ

### 基本仕様

| 項目 | 値 |
|------|-----|
| 設置単位 | 1ブロック |
| 自動接続 | 隣接パイプ/機械に自動接続 |
| 形状 | 直線/L字/T字/クロス自動判定 |
| 混合 | 不可（1種類の液体のみ） |

### ティア

| ティア | 流量 | 特徴 |
|--------|------|------|
| Tier 1 | 100 mB/s | 基本 |
| Tier 2 | 300 mB/s | 中盤 |
| Tier 3 | 1000 mB/s | 大容量 |

### 高さ制限

| 条件 | 制限 |
|------|------|
| 上り | 10ブロックまで（ポンプで延長） |
| 下り | 制限なし |
| 水平 | 制限なし |

## ポンプ

パイプの揚程を追加。

| 項目 | 値 |
|------|-----|
| 追加揚程 | +10ブロック |
| 消費電力 | 10kW |
| サイズ | 1×1×1 |

**設置:** パイプラインの途中に挿入

## タンク

液体の貯蔵。

| 項目 | 値 |
|------|-----|
| 容量 | 10,000 mB（10バケツ） |
| サイズ | 1×1×2 |
| 入出力 | 上面:入力、下面:出力、側面:両方 |

### マルチブロックタンク

隣接配置で容量増加。

| 配置 | 容量 |
|------|------|
| 1個 | 10,000 mB |
| 2×2×2 (8個) | 80,000 mB |
| 3×3×3 (27個) | 270,000 mB |

## 液体の種類

### 基本液体

| 液体 | 用途 |
|------|------|
| 水 | 冷却、蒸気生成、農業 |
| 溶岩 | 熱源、一部レシピ |
| 原油 | 精製して燃料に |
| 蒸気 | タービン発電 |

### 加工液体

| 液体 | 生成 | 用途 |
|------|------|------|
| 燃料 | 原油精製 | 燃料発電機 |
| 潤滑油 | 原油精製 | 高速機械の必須 |
| 硫酸 | 化学プラント | バッテリー製造 |
| 冷却液 | 化学プラント | 原子炉冷却 |

## ガス

気体の処理。パイプで輸送可能。

| ガス | 生成 | 用途 |
|------|------|------|
| 蒸気 | ボイラー | タービン発電 |
| 天然ガス | 採掘 | 発電、化学原料 |
| 酸素 | 電気分解 | 宇宙ステーション |
| 水素 | 電気分解 | 燃料電池 |

## 液体を扱う機械

| 機械 | 入力 | 出力 |
|------|------|------|
| ポンプ（採掘） | - | 水/原油/溶岩 |
| ボイラー | 水 + 燃料 | 蒸気 |
| 精油所 | 原油 | 燃料 + 潤滑油 |
| 化学プラント | 各種 | 各種 |
| タービン | 蒸気 | 電力 + 水 |

## バルブ

パイプの流れを制御。

| 種類 | 機能 |
|------|------|
| 遮断バルブ | 手動/信号でON/OFF |
| 逆止弁 | 一方向のみ流れる |
| 排水バルブ | 余剰液体を廃棄 |
| 優先バルブ | 指定方向を優先 |

## 接続制御

レンチでパイプの各面の接続をON/OFF。

```
    [機械A]
       │ ← この接続をOFF
    [パイプ]──[パイプ]──[機械B]
```

## 液体の混合防止

| 状況 | 動作 |
|------|------|
| 異なる液体が合流 | 後から来た液体は流れない |
| 空パイプへの投入 | 最初に入った液体のみ受け入れ |
| パイプの洗浄 | レンチで「洗浄」操作（内容物破棄） |

## 設計思想

| ポイント | 理由 |
|----------|------|
| 混合不可 | 事故防止、シンプル |
| ブロック単位設置 | コンベアと操作統一 |
| 揚程制限 | ポンプの存在意義 |
| マルチブロックタンク | 大容量需要への対応 |
