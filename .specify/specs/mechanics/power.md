# 電力システム

## 概要

電気エネルギーのみの単一システム。シンプルで管理しやすい。

~~運動エネルギー（シャフト）は削除。~~
水車・風車は見た目として回転するが、出力は電気。

## 電力ネットワーク

| 項目 | 仕様 |
|------|------|
| 単位 | kW（キロワット） |
| 伝達 | ワイヤーで接続 |
| ネットワーク | BFS探索で接続判定 |
| 損失 | なし（シンプル化） |

## 過負荷時の動作

| 状態 | 消費/発電比 | 動作 |
|------|-------------|------|
| 正常 | 〜80% | 全機械100%稼働 |
| 警告 | 80〜100% | 警告表示、正常稼働 |
| 過負荷 | 100%超 | 全機械が速度低下（グレースフルデグレード） |

**過負荷時:** 発電量 ÷ 消費量 の比率で全機械の速度が低下。
例: 200kW発電、300kW消費 → 全機械が66%速度で稼働

## 発電機

| 発電機 | 発電量 | 燃料 | 特徴 |
|--------|--------|------|------|
| 水車 | 10kW | 水流 | 序盤、見た目が回転 |
| 風車 | 20kW | 風（高度） | 序盤、見た目が回転 |
| バイオマス発電機 | 30kW | バイオマス | 序盤〜中盤 |
| 石炭発電機 | 75kW | 石炭 | 中盤 |
| 燃料発電機 | 250kW | 燃料（液体） | 中盤〜終盤 |
| ソーラーパネル | 30kW | 日光 | 昼のみ |
| 原子力発電機 | 2,500kW | ウラン燃料棒 | 終盤 |

|------|-----|

## 導管システム（EnderIO風）

電力・液体・信号を1ブロックにまとめて設置可能。

### 複合設置

```
1つのブロック空間に同時設置可能:
├─ 電力ケーブル
├─ 液体パイプ
└─ 信号ケーブル
```

配線スペースを節約し、工場をすっきり設計できる。

### 電力ケーブル

| 項目 | 仕様 |
|------|------|
| 設置単位 | 1ブロック |
| 自動接続 | 隣接ケーブル/機械に自動接続 |
| 伝達距離 | 無制限 |
| 損失 | なし |
| 接続制御 | レンチでその面のON/OFF |
| 複合設置 | 液体・信号と同じブロックに設置可能 |

### 種類

| 種類 | 機能 |
|------|------|
| 基本ケーブル | 直線/L字/T字/クロス自動判定 |
| 被覆ケーブル | 隣接しても接続しない（分離用） |
| ファサード | 他ブロックの見た目で隠せる |

## 電力制御

| 機械 | 機能 |
|------|------|
| サーキットブレーカー | 過負荷時に自動遮断、手動リセット |
| パワースイッチ | 手動ON/OFF、信号連動可能 |

## 設計思想

**なぜ電気のみか？**

| 理由 | 詳細 |
|------|------|
| シンプル | 二系統管理は複雑、ストレスフリーに反する |
| スケーラブル | 大規模工場でも管理しやすい |
| 配線自由度 | ワイヤーは物理制約が少ない |

**水車・風車の役割:**
- 序盤の無燃料発電として有用
- 見た目が動いて楽しい（視覚的報酬）
- 内部的には電気なので複雑さなし
