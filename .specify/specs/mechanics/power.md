# 電力システム

## 概要

運動エネルギー（シャフト）と電気エネルギー（ワイヤー）の二系統。

## 運動エネルギー（シャフト）

Minecraft Create Mod風の回転力伝達。

| 項目 | 仕様 |
|------|------|
| 伝達方式 | トルク × 回転速度 |
| 逆回転 | 破損（ギア比で調整） |
| ギア比 | 速度とトルクのトレードオフ |

### 発生源

- 水車（水流）
- 風車（高度）
- 蒸気エンジン

### 消費先

- 粉砕機
- プレス機
- ポンプ

## 電気エネルギー（ワイヤー）

従来の工場ゲーム風の電力ネットワーク。

| 項目 | 仕様 |
|------|------|
| 伝達方式 | 電力量（kW） |
| ネットワーク | BFS探索で接続判定 |
| 過負荷 | グレースフルデグレード（速度低下） |

### 発生源

- 発電機（蒸気タービン）
- ソーラーパネル
- 燃料発電機
- 原子力発電機（終盤）

### 消費先

- 組立機
- 化学プラント
- ロボットポート

## 熱システム

| 項目 | 仕様 |
|------|------|
| 熱源 | 燃焼、原子炉 |
| 熱伝導 | ホットタンク必須 |
| 冷却 | 冷却塔、水冷 |

## 設計思想

**なぜ二系統か？**

- 序盤は運動エネルギー（シンプル、視覚的）
- 中盤以降は電気（スケーラブル）
- 選択肢を増やしてプレイスタイルの多様性を確保
