### 粉砕機 (Crusher)

鉱石を粉砕して粉にする。粉から精錬すると効率UP。

| 項目 | Tier 1 | Tier 2 | Tier 3 |
|------|--------|--------|--------|
| 処理速度 | 30/分 | 60/分 | 120/分 |
| 消費電力 | 10kW | 30kW | 100kW |
| サイズ | 1×1×1 | 1×1×1 | 2×2×2 |

**レシピ例:**
| 入力 | 出力 | 効果 |
|------|------|------|
| 鉄鉱石×1 | 鉄の粉×2 | 精錬で鉄インゴット×2（2倍） |
| 銅鉱石×1 | 銅の粉×2 | 精錬で銅インゴット×2（2倍） |
| 石×1 | 砂利×1 | 建材用 |

## 共通仕様

### オーバークロック

全ての加工機械に適用可能。

| 倍率 | 消費電力 | 必要パワーシャード |
|------|----------|-------------------|
| 100% | 1× | 0 |
| 150% | 1.5^1.6 ≈ 1.9× | 1 |
| 200% | 2^1.6 ≈ 3× | 2 |
| 250% | 2.5^1.6 ≈ 4.3× | 3 |

### 品質システム

| 品質 | 確率（基本） | 効果 |
|------|-------------|------|
| Normal | 100% | なし |
| Uncommon | - | 効率+10% |
| Rare | - | 効率+25% |
| Epic | - | 効率+50% |
| Legendary | - | 効率+100% |

品質モジュールで高品質アイテム生成確率UP。

### モジュールスロット

機械によって0〜6スロット。
モジュールの種類: 速度、効率、品質
