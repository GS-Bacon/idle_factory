# 進行・アンロックシステム

## 概要

クエスト達成を通じて機械・レシピ・機能がアンロックされる。
「研究」や「テクノロジーツリー」は採用しない（シンプル化）。

## アンロック方式

| 方式 | 採用 | 理由 |
|------|------|------|
| クエスト報酬 | ✓ | メインの進行軸 |
| 研究/テックツリー | ✗ | 複雑、ストレス |
| クラフト解放 | ✗ | 素材集めの手間 |
| 探索発見 | △ | サブクエストで誘導 |

## ティア構成

| ティア | 内容 | 解放条件 |
|--------|------|----------|
| Tier 0 | 初期装備（ピッケル、精錬炉） | ゲーム開始時 |
| Tier 1 | 基本自動化（採掘機、コンベア） | クエスト報酬 |
| Tier 2 | 金属加工（プレス機、粉砕機） | クエスト報酬 |
| Tier 3 | 電力導入（発電機、ワイヤー） | クエスト報酬 |
| Tier 4 | 高度加工（組立機、化学プラント） | クエスト報酬 |
| Tier 5 | 自動化拡張（ロボット、ブループリント） | クエスト報酬 |
| Tier 6 | 大規模輸送（列車、パイプライン） | クエスト報酬 |
| Tier 7 | 終盤（原子力、宇宙ステーション部品） | クエスト報酬 |

## ティア別詳細

### Tier 0: 初期

| 項目 | 内容 |
|------|------|
| 機械 | 精錬炉×1 |
| ツール | ピッケル |
| レシピ | 鉄鉱石→鉄インゴット、銅鉱石→銅インゴット |

### Tier 1: 基本自動化

| 項目 | 内容 |
|------|------|
| 機械 | 採掘機、コンベア（分岐・スプリッター・フィルター含む） |
| 解放クエスト | 「最初の一歩」完了 |
| できること | 採掘の自動化 |
| 備考 | 物流機能は個別ロックしない（コンベア入手で全機能使用可） |

### Tier 2: 金属加工

| 項目 | 内容 |
|------|------|
| 機械 | プレス機、粉砕機 |
| レシピ | 鉄板、銅ワイヤー、鉄パイプ |
| 解放クエスト | 「金属加工」完了 |
| できること | 加工品の生産 |

### Tier 3: 電力導入

| 項目 | 内容 |
|------|------|
| 機械 | 石炭発電機、ワイヤー、水車 |
| 解放クエスト | 「電気の時代」完了 |
| できること | 電力を使う機械が稼働可能に |

### Tier 4: 高度加工

| 項目 | 内容 |
|------|------|
| 機械 | 組立機、ミキサー、化学プラント |
| レシピ | 電子部品、回路基板、プラスチック |
| 解放クエスト | 「複合生産」完了 |
| できること | 複数素材からの製品生産 |

### Tier 5: 自動化拡張

| 項目 | 内容 |
|------|------|
| 機械 | ロボポート、物流ロボット、建設ロボット |
| 機能 | ブループリント |
| 解放クエスト | 「ロボット工学」完了 |
| できること | 物流自動化、設計の保存・再利用 |

### Tier 6: 大規模輸送

| 項目 | 内容 |
|------|------|
| 機械 | 列車、レール、駅、大型タンク |
| 機能 | 液体パイプライン |
| 解放クエスト | 「物流革命」完了 |
| できること | 長距離大量輸送 |

### Tier 7: 終盤

| 項目 | 内容 |
|------|------|
| 機械 | 原子力発電機、宇宙ステーション部品製造機 |

| できること | 最終目標への道 |

## 機械のアップグレード

同じ機械でもティア違いが存在。

| 機械 | Tier 1 | Tier 2 | Tier 3 |
|------|--------|--------|--------|
| 採掘機 | 30/分 | 60/分 | 120/分 |
| コンベア | 60/分 | 120/分 | 270/分 |
| 精錬炉 | 1× | 1.5× | 2× |

### アップグレード方法

| 方法 | 説明 |
|------|------|
| 置き換え | 新しいティアの機械を同じ場所に設置 |
| その場アップグレード | レンチ + アップグレードキットで変換 |

## 設計思想

| ポイント | 理由 |
|----------|------|
| クエスト報酬でアンロック | 進行が明確、迷わない |
| 研究システムなし | 複雑さを排除、すぐ使える |
| 段階的解放 | 情報過多を防ぐ、学習曲線を緩やかに |
| アップグレードは置き換え | 既存ラインを活かせる |
