# コンベア

## 概要

アイテムを自動輸送する基本物流機械。

## 仕様

| 項目 | 値 |
|------|-----|
| レーン | 両側レーン（左右独立） |
| 速度ティア | 複数段階（Tier 1-4など） |

## 合流動作

| 種類 | 動作 |
|------|------|
| ジッパー合流 | 左右交互にアイテムを受け入れ |
| 優先合流 | 指定方向を優先 |

## 分配動作

| 種類 | 動作 |
|------|------|
| ラウンドロビン | 出力先を順番に切り替え |
| 均等分配 | 各出力に均等に振り分け |
| 優先度分配 | 指定出力を優先、溢れたら次へ |

## 詰まり検出

- 出力先が満杯の場合、コンベア上で詰まり発生
- 詰まり状態を視覚的に表示
- 警告システムと連携

## 両側レーン

- 左レーンと右レーンが独立
- サイドローディング対応（横から特定レーンに投入）
- レーン指定分配が可能
