| 項目 | 内容 |
|------|------|
| 初期装備 | Miner×3, Conveyor×10, Crusher×2, Furnace×2, IronOre×5, Coal×5 |
| 操作説明 | チュートリアルポップアップ（起動時に表示） |
| Quest 1 | 鉄インゴット3個納品 → 採掘機×2, コンベア×20 |
| Quest 2 | 鉄インゴット100個納品 → 採掘機×2, コンベア×40, 精錬炉×2 |
| Quest 3 | 銅インゴット50個納品 → 粉砕機×2, 精錬炉×2 |
