# .specify

仕様書・開発ガイド格納フォルダ

## 構造

```
.specify/
├── specs/           # ゲーム仕様
│   ├── index.md     # ← 入り口
│   ├── core-concept.md
│   ├── editor.md
│   ├── first-30-minutes.md
│   ├── mvp-v0.1.md
│   └── ui.md
├── memory/          # 開発ガイド
│   ├── bevy-tips.md
│   ├── input-rules.md
│   ├── magicavoxel-guide.md
│   ├── modeling-rules.md (VOX方式)
│   └── ui-design-rules.md
└── roadmap.md       # ロードマップ
```

## 読み方

1. **仕様を知りたい** → `specs/index.md`
2. **実装ガイド** → `memory/` 内の該当ファイル
3. **将来計画** → `roadmap.md`
