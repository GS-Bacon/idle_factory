# Idle Factory

## このプロジェクトについて

**バイブコーディング限界実験** - 非エンジニアがAIだけで大規模ゲームを作れるか？

### 実験概要

| 項目 | 内容 |
|------|------|
| 開発者 | 非エンジニア（プログラミング経験なし） |
| 開発手法 | Claude Code によるAI駆動開発 |
| 期間 | 2025年〜 |
| 目標 | Factorio風の本格的な工場自動化ゲーム |

### 現在の規模

| 指標 | 値 |
|------|-----|
| コード行数 | 34,000+ 行 |
| テスト数 | 700+ 件 |
| バージョン | 0.3.136 |

## ゲーム概要

3Dボクセル工場自動化ゲーム。採掘機でリソースを収集し、コンベアで運搬、精錬炉や粉砕機で加工して、クエストをクリアしましょう。

### 特徴

- **ストレスフリー設計** - 敵なし、HP/空腹なし、落下ダメージなし
- **自動化ファースト** - 機械とコンベアで全自動生産ライン構築
- **Minecraft風操作** - WASD移動、一人称視点、ブロック配置

## 遊び方

### ダウンロード

[Releases](https://github.com/bacon-and-and-and/idle_factory/releases) からお使いのOS用のファイルをダウンロード

| OS | ファイル |
|----|----------|
| Windows | `idle_factory-windows.zip` |
| Linux | `idle_factory-linux.tar.gz` |
| macOS | `idle_factory-macos.tar.gz` |

### 起動

ダウンロードしたファイルを展開して `idle_factory` を実行

## 操作方法

| キー | 操作 |
|------|------|
| W / A / S / D | 移動 |
| マウス | 視点操作 |
| 左クリック | ブロック破壊 |
| 右クリック | ブロック/機械設置、機械UI開く |
| 1 - 9 | ホットバー選択 |
| E | インベントリ開閉 |
| Q | クエスト報酬受け取り |
| ESC | ポーズメニュー |

## ゲームの流れ

1. **採掘機を設置** - 鉱石の上に置くとアイテムを自動生成
2. **コンベアで運搬** - 採掘機から出たアイテムを運ぶ
3. **精錬炉/粉砕機で加工** - 鉱石をインゴットに変換
4. **納品プラットフォームへ** - 緑色のプラットフォームに納品
5. **クエストクリア** - 報酬を獲得して次のステージへ

## バイブコーディングの記録

このプロジェクトでは以下を記録しています：

- AIとの対話ログ（WORK_LOG.md）
- 破綻パターンと回復方法
- 人間が介入した判断

### 観測している破綻パターン

1. **コンテキスト限界** - AIが以前の決定を忘れる
2. **メタワーク暴走** - ドキュメントだけ増えてゲームが進まない
3. **修正ループ** - AIがAIのコードを直す無限ループ
4. **設計の不整合** - モジュール間の噛み合わせ問題
