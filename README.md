# Idle Factory

ボクセルベースの工場建設ゲームです。採掘機でリソースを収集し、コンベアで運搬、精錬炉や粉砕機で加工して、クエストをクリアしましょう。

（準備中）

## ビルド方法

### 必要な環境

- Rust (2021 edition)
- Bevy 0.15

### ネイティブビルド

```bash
cargo build --release
```

### WASMビルド

```bash
./build-wasm.sh
```

## 起動方法

### ネイティブ版

```bash
# 推奨（RDPディスプレイ自動検出）
./run.sh

# または直接実行
cargo run --release
```

### WASM版

```bash
# ビルドとサーバー起動
./deploy-wasm.sh

# ブラウザでアクセス
# ローカル: http://10.13.1.1:8080
# Tailscale: http://100.84.170.32:8080
```

## 操作説明

| キー | 操作 |
|------|------|
| W / A / S / D | 移動（前後左右） |
| マウス | 視点操作 |
| 左クリック | ブロック破壊 |
| 右クリック | ブロック/機械設置 |
| 1 - 9 | ホットバー選択 |
| E | 精錬炉UIを開く |
| Q | クエスト報酬受け取り |
| ESC | カーソル解放 |

## ゲーム目標

1. **採掘機を設置** - 地面に置くと下のブロックに応じたリソースを無限に生成します
2. **コンベアで運搬** - 採掘機から出たアイテムを運びます
3. **精錬炉/粉砕機で加工** - 鉄鉱石を鉄インゴットに精錬、銅鉱石を粉砕して銅インゴットに加工
4. **納品プラットフォームに納品** - 緑色のプラットフォームにアイテムを運び込みます
5. **クエストをクリア** - 指定されたアイテムを納品して報酬を獲得しましょう
